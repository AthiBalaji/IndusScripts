Information

Each text has an identifying 10 digit text code based on the original book "Texts, Concordance and Values". With each digit in representing a particular detail of the text. The first four digit of the text can be used to identify texts and if the search field can be used to search the complete collection of texts using the four digit number associated with it. For more information, read the book "Texts, Concordances and Values"
