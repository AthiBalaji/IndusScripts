About the website

The website consists of the complete collection Indus-Valley texts and symbols detailed in the book "Texts, Concordance and Tables" by Iravatham Mahadevan with support for search operations, details on the concordance of each sign that make up a given text and a breakdown of the texts using a graph theoretical approach with various centrality measures. The website supports: Search engine for the collection of texts with capacity to search and display a given text, Analytics with frequency of each sign and a display of the complete list of texts that contain it and Analysis of the texts using a graph theoretical approach

About Indus scripts

The Indus valley civilization is one of the oldest civilizations in the world and efforts to decipher the written language left behind by its inhabitants have not succeeded till date. It's decipherment would a monumental achievement of great historical and archaelogical value and this website which is done as a four-week project is done to assist that cause.
